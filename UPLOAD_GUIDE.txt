새 저장소에 올릴 프로젝트입니다.

중요:
1. 최상위 파일들을 모두 업로드합니다.
2. GitHub 웹 업로드에서 .github 폴더가 빠지면,
   Add file → Create new file로 다음 경로를 만듭니다.
   .github/workflows/update.yml
3. 최상위 WORKFLOW_UPDATE.yml 내용을 복사해 붙여넣습니다.
4. Settings → Pages → Source를 GitHub Actions로 선택합니다.
5. Actions → Sports schedule update → Run workflow를 실행합니다.
